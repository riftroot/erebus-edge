# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities to **contact@riftroot.com** rather than opening a public GitHub issue. Do not disclose security issues publicly until they have been assessed and addressed.

## Scope

- This documentation repository and the riftroot.com site.
- The Erebus Edge implementation is closed-source. Coordinated disclosure for any vulnerabilities inferred from architectural documentation is welcome.

## Response SLA

We will acknowledge receipt within **72 hours** and provide an assessment timeline as soon as possible.

## Bug Bounty

There is no bug bounty program at this time. Credit in a public acknowledgement is available on request.

## Notes

- **Do NOT open public issues** for security reports.
- For general inquiries: contact@riftroot.com
